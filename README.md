# Vue 3 Study Lab

A comprehensive showcase of modern Vue.js development practices, featuring the latest tools and libraries used in the market. This project serves as a laboratory for experimenting with Vue 3 ecosystem technologies.

## üöÄ Features

- **Modern Tooling**: Built with Vite 4 and TypeScript 5.
- **State Management**: Powered by Pinia.
- **Data Fetching**: Server state management with TanStack Query (Vue Query).
- **Form Validation**: Robust form handling using VeeValidate and Zod.
- **UI Components**: Reusable components built with Tailwind CSS.
- **Feature Showcase**:
    - **Todo List**: Full CRUD with filtering.
    - **Calendar**: Interactive month view.
    - **Data Table**: Sorting, pagination, and search.
    - **Charts**: Visualizations using Chart.js.
